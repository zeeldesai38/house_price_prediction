# House Price Prediction using Linear Regression

## Project Overview
This project predicts house prices using Linear Regression based on features such as:

- Bedrooms
- Bathrooms
- SUVs
- Cars
- Garages
- Domestic Help Rooms
- Age of House
- House Area
- House Category

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Workflow

1. Data Cleaning
2. Missing Value Handling
3. Feature Encoding
4. Train-Test Split
5. Linear Regression
6. Model Evaluation


## Conclusion

The Linear Regression model achieved an R² score of 84.32%, demonstrating good predictive performance for house price estimation.
